# Generated art credits

All images in this folder are AI-generated on a Microsoft Foundry deployment
of **MAI-Image-2e** by the repo maintainers, using the prompts in
[tools/generate_art.py](../../../tools/generate_art.py). They are original
outputs (no third-party art inputs) and ship under this repo's MIT license.

The narration takes under `lore/narration/` are AI-generated speech from an
Azure **gpt-4o-mini-tts** deployment, baked with
[tools/generate_narration.py](../../../tools/generate_narration.py) from the
script in [ui/game/intro.js](../../game/intro.js). Same terms: original
outputs, MIT.

Disclosure: AI-generated content. Models: MAI-Image-2e (images),
gpt-4o-mini-tts (narration audio).
